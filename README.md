# Telegram Bot MarvelCharm

### Возможности:
- Баланс и реферальная система
- Ежедневный бонус
- Топ-лидеры по рефералам
- Админ-панель: добавление баланса, рассылка, топ лидеров

### Установка:
1. Клонировать репозиторий
git clone <repo_url>

2. Установить зависимости:


pip install -r requirements.txt

3. Настроить переменные окружения в Render:
- BOT_TOKEN
- ADMINS
- START_BALANCE
- REF_BONUS
- DAILY_BONUS
4. Запустить бота:


python bot.py
